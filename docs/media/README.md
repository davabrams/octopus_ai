# README media

Drop the captures listed below into this folder using **exactly these filenames**
— the README references them by path. Suggested size ~1200–1600 px wide for
stills; keep GIFs/MP4s short (5–12 s) and reasonably compressed.

| File | Type | What to capture |
|------|------|-----------------|
| `hero.gif` | GIF / MP4 | The headline shot: browser analyzer **Playback** of the octopus crawling and camouflaging over the image surface, arms moving. ~8 s loop. |
| `analyzer.png` | PNG | Full browser analyzer window — canvas + Runs list + a sucker/agent inspector panel visible. |
| `camouflage.png` | PNG | Zoomed-in suckers matching the surface colors (color mode "after"), ideally over a high-contrast patch. |
| `ilqr-arms.png` | PNG | An iLQR run with the **iLQR overlay on** — arms reaching/fleeing with the ghost horizon trajectories (yellow) and red centerlines. |
| `exploration.png` | PNG | Playback with the **exploration** overlay on — the amber recency heatmap of visited cells. |
| `cost-panel.png` | PNG | The per-node cost inspector panel (hover/select a node) showing spring / bending / attract / repel / effort with the direction arrows. |
| `pursuit-flee.gif` | GIF / MP4 | Threats (pink) pursuing and prey (green) fleeing; the octopus scrunching away from a threat. ~6 s. |
| `matplotlib.png` | PNG | The native matplotlib viewer window mid-run (with force arrows if using the `VIZ` profile). |
